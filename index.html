MSTN Mutation via GAN - Demo

Through the use of comptational biology and machine learning algorithms, we aim to generate custom MSTN mutations that will result in improved myostatin inhibition to ultimately treat Muscle Atrophy using a CRISPR or a targted phramalogical approach.


How Would the ML Algorithm Work?

The input data passed into the network would be the 506 BP long sequence for the MSTN unmutated gene. The nucleotoides of this gene will be autoencoded using a hashing function instead of using a one-hot vector to reduce the dimenosionality of the problem. The input data is then convoled through the first set of layers of the Generative Adversarial Network (GAN) until it reaches the intermeidate layer. Following this, the discrimantor function is used to ensure that the current permutation of the Gene sequence is an improvement on the previous version. This process is repeated iteratively thousands of times so that each sequence generated after that point is unique and effective.

Conclusions

Currently, we are in the process of creating both the GAN Machine Learning Model as well as the custom discriminator function so as to determine an optimal mutated version of the MSTN gene. Following this, we plan on performing an in vitro study with the mutated gene to quantitatively asses whether or not the gene is able to inhibit myostatin production. Lastly, we aim to created a targeted phisiological approach with the information acquired from this gene in order to effectively treat Muscle Atrophy in patients.
